# Description
- Push_swap은 - **두 개의 스택**과 **정수 리스트**, 그리고 제한된 명령어 집합을 사용하여 데이터를 정리하는 알고리즘을 만드는 과제입니다. 

- **과제의 목표**는`push_swap`이라는 C 프로그램을 작성해 주어진 정수들을 오름차순으로 정렬하는 데 필요한 **최단 명령어 시퀀스**를 계산하고 출력하는 것입니니다. 

# Instructions
- 소스 파일들을 컴파일할 `Makefile`을 제출해야합니다. `relink`는 허용되지 않습니다.
- 'Makefile'에는 $(NAME), all, clean, fclea과 re라는 rule이 존재해야합니다. 
- 'bonus'를 제출하기 위해서는 'Makefile'에 **bonus*** rule이 포함되어야합니다. 
- 'bonus'를 위한 파일들은 *_bonus.{c/h}라는 파일명을 가지고 있어야합니다. 
- libft를 사용하기 위해서는 해당 파일들과 Makefile을 포함하고 있어햐며, 프로젝트의 Makefile은 libft의 Makefile을 사용하여 라이브러리를 컴파일 한 다음, 프로젝트를 컴파일 해야합니다. 
- 실행
	- root 디렉토리에서 make 명령어 실행.
	- push_swap 프로그램을 여러가지 테스트 케이스를 이용해서 실행.
	- bonus 평가를 위하여, make bonus 명령어를 실행.
